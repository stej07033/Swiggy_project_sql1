# Swiggy_project_sql1
Swiggy SQL Data Analysis Project – A PostgreSQL project that analyzes Swiggy restaurant data using SQL queries. Covers data filtering, sorting, aggregation, string functions, CASE statements, and business-oriented insights from restaurant data.
